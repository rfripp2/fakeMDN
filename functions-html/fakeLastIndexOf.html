fakeLastIndexOf retorna el ultimo indice de un array donde se encuntre el elemento que pasemos como parametro,en caso que no exista,retorna -1.

Demo

          
            let animales = ["perro","gato","vaca","perro","caballo"];

            fakeLastIndexOf(animales,"perro");
            // 3

Sintaxis

fakeLastIndexOf(array,element)

Par√°metros
array
El array que la funcion recorrera
element
el elemento que la funcion buscara el ultimo indice donde se encuentre

Descripcion

fakeLastIndexOf, retorna el ultimo indice donde se encuentre el elemento, usando comparacion estricta.

Ejemplos

Buscar el color amarillo


                   let colores = ["azul","gris","negro"];

                   fakeLastIndexOf(colors,"amarillo");
                   // -1
