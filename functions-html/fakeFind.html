fakeFind function

La funcion fakeFind retorna el valor del primer elemento que pase el test de la funcion que se ejecute.

Demo

fakeFind([3, 65, 100, 1000], item => item > 50);
            
// 65

Sintaxis

fakeFind(array,callback(currentValue));

Par√°metros

array

El array que la funcion recorrera

callback

La funcion que se ejecutara para cada item del array.

Parametro 3

  • currentValue

    • Cada item del array que sera procesado por la funcion callback.

Valor que retorna

El valor del primer elemento que pase el test de la funcion provista.

Descripcion

fakeFind llama a la funcion callback,que se ejecutara para cada elemento hasta que la funcion retorne true,y retornara el valor de dicho elemento.
Si ningun elemento pasa el test, retornara undefined.

Ejemplos

Retorna el primer item con longuitud > 25 characteres que encuentra en el array. 


            let carreraDeCharacteres = [
            "tengo algunos caracteres",
            "yo no",
            "yo tendre mas de 20?!!! espero ser el primero que pasa el test",
            "yo seguro tengo mas que 20 ,pero capaz ya me ganaron"
          ];
            fakeFind(carreraDeCharacteres,item => item.length > 25);
              
//"yo tendre mas de 20?!!! espero ser el primero que pasa el test"
