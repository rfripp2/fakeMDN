_union() crea un array con valores unicos que concatenando los 2 arrays. Se utiliza la regla SameValueZero para la comparacion.

Demo

          
                [1, 1, 1, 2, 3, 4]._union([1, 1, 4, 5, 8, 9]);
                //[1,2,3,4,5,8,9];

Sintaxis

arr._union(arr2)
