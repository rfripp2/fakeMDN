El metodo _filter crea un nuevo array con los elementos que pasen el test de la funcion proveida en el callback.

Demo

          
            [3, 65, 100, 1000]._filter(number => number > 65);
            
// [100,1000]

Sintaxis

arr._filter(callback(elemento))

Par√°metros
callback
La funcion que se ejecutara para cada elemento del array

  • elemento

El elemento que sera procesado por la funcion callback

Descripcion

_filter llama a la funcion callback,la cual testea si da true o false para cada elemento del array. Los que pasen el test,(la funcion retorne true),seran copiados a un nuevo array,el cual se retornara. Si ningun elemento pasa el test, se devuelve un array vacio.

_filter no muta el array

Ejemplos

Filtrar numeros positivos 


            let numbers = [-4,3,7,-100];
            numbers._filter(number => number > 0);
            // [3,7];
