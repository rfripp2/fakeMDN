El metodo _forEach ejecuta una funcion para cada elemento de un array

Demo

          
          let numbers = [1,2,3]
            numbers._forEach(element => console.log(element));
            
//1
            
//2
            
//3

Sintaxis

_forEach(callback(elemento))

Par√°metros
callback
La funcion que se ejecutara para cada elemento del array

  • elemento

El elemento que sera procesado por la funcion callback

Descripcion

_forEach ejecuta la funcion callback, que sera ejecutada para cada elemento del array que se provea. _forEach no retorna ni muta el array, salvo que se especifique en el callback que queremos retornar algun valor en algun binding.

Ejemplos

Confirmar que no muta el array.

let numbers = [1,2,3];
            
numbers._forEach(element => {
              element *= 2;
              console.log(element);
            });
            
// 2
            
// 4
            
// 6
           
console.log(numbers);
           
// [1,2,3];
