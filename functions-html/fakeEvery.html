Funcion fakeEvery()

El método fakeEvery() prueba si todos los elementos del array pasan la prueba implementada por la función proporcionada. Devuelve un valor booleano.

Sintaxis

fakeEvery(array,callback(elemento))

Parámetros

array

El array que la funcion recorrera

callback

La funcion que se ejecutara para cada elemento del array

  • elemento

El elemento que sera procesado por la funcion callback

Descripcion

La función fakeEvery ejecuta la función callback dada una vez por cada elemento presente en el arreglo hasta encontrar uno que haga retornar un valor false a callback (un valor que resulte false cuando se convierta a booleano). Si no se encuentra tal elemento, el método every de inmediato retorna false. O si callback retorna true para todos los elementos, fakeEvery retornará true.
