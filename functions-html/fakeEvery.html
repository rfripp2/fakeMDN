Funcion fakeEvery()

El método fakeEvery() prueba si todos los elementos del array pasan la prueba implementada por la función proporcionada. Devuelve un valor booleano.

Demo

          
                function fakeMap(array, callback) {
                    const arrayMapped = [];
                    fakeForEach(array, element => {
                    arrayMapped.push(callback(element));
                    });
                    return arrayMapped;
                }

Sintaxis

fakeEvery(array,callback(elemento))

Parámetros
array
El array que la funcion recorrera
callback
La funcion que se ejecutara para cada elemento del array

  • elemento

El elemento que sera procesado por la funcion callback

Ejemplos

Evaluar si los numeros del array cumplen con la condición

fakeEvery([1, 23, 76, 45], numbers => numbers < 55); // false

Evaluar si los valores del array cumplen con la condición

fakeEvery(["jose", "ana"; "antonio", "maria"], elements => elements!= "julia"); // true
