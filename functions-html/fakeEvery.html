Funcion fakeEvery()

El método fakeEvery() prueba si todos los elementos del array pasan la prueba implementada por la función proporcionada. Devuelve un valor booleano.

Demo

          
            function fakeEvery(arr, callback) {
              let trusthyArr = [];
              fakeForEach(arr, element => {
                if (callback(element)) {
                  trusthyArr.push(element);
                }
              });
              return trusthyArr.length == arr.length;
            }

Sintaxis

fakeEvery(array,callback(elemento))

Parámetros
array
El array que la funcion recorrera
callback
La funcion que se ejecutara para cada elemento del array

  • elemento

El elemento que sera procesado por la funcion callback

Descripcion

El método fakeEvery ejecuta la función callback dada una vez por cada elemento presente en el arreglo hasta encontrar uno que haga retornar un valor falso a callback (un valor que resulte falso cuando se convierta a booleano). Si no se encuentra tal elemento, el método every de inmediato retorna false. O si callback retorna trueverdadero para todos los elementos, every retornará true. callback es llamada sólo para índices del arreglo que tengan valores asignados; no se llama para índices que hayan sido eliminados o a los que no se les haya asignado un valor.

Ejemplos

Evaluar si los numeros del array cumplen con la condición

fakeEvery([1, 23, 76, 45], numbers => numbers < 55); // false

Evaluar si los valores del array cumplen con la condición

fakeEvery(["jose", "ana"; "antonio", "maria"], elements => elements!= "julia"); // true
