La función reduce() aplica una función a un acumulador y a cada valor de un array (de izquierda a derecha) para reducirlo a un único valor.

Demo

          
            var numbers = [0, 1, 2, 3]
            function sum(a, b){ 
                return a + b; 
            }
            fakeReduce(numbers, sum);
            // total == 6

Sintaxis

fakeReduce(array, callback(elemento))

Parámetros
array
El array que la función recorrera.
callback
La función que se ejecutara para cada elemento del array.

  • elemento

El elemento que sera procesado por la función callback.

Descripción

fakeReduce ejecuta la función una vez por cada elemento presente en el array, excluyendo los huecos del mismo.

Ejemplos

Acumular cuadrados

            
                var array1 = [1, 2, 3, 4];
                function square(a, b){ 
                    return (a * a) + (b * b); 
                  }
                fakeReduce(array1, square);
                // total == 30

Evaluar si los valores del array cumplen con la condición

            
                var integrado = [[0,1], [2,3], [4,5]]
                function concatenar(a,b){
                    return a.concat(b);
                }
                fakeReduce(integrado, concatenar);
                // integrado es [0, 1, 2, 3, 4, 5]
