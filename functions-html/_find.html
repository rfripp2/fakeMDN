El metodo _find() retorna el valor del primer elemento que pase el test de la funcion que se ejecute.

Demo

          
            [3, 65, 100, 1000]._find(item => item > 50);
            
// 65

Sintaxis

arr.find(callback(elemento)

Par√°metros
callback
La funcion que se ejecutara para cada elemento del array

  • elemento

El elemento que sera procesado por la funcion callback

Descripcion

_find llama a la funcion callback,que se ejecutara para cada elemento hasta que la funcion retorne true,y retornara el valor de dicho elemento.
Si ningun elemento pasa el test, retornara undefined.

Ejemplos

Retorna el primer item con longuitud > 25 characteres que encuentra en el array. 

let carreraDeCharacteres = [
            "tengo algunos caracteres",
            "yo no",
            "yo tendre mas de 20?!!! espero ser el primero que pasa el test",
            "yo seguro tengo mas que 20 ,pero capaz ya me ganaron"
          ];
            carreraDeCharacteres._find(item => item.length > 25);
              
//"yo tendre mas de 20?!!! espero ser el primero que pasa el test"
