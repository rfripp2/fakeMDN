fakeArrayMin retorna el valor numerico mas chico que encuentre en un array.

Demo

          
            [1, 2, 3, -4, 5]._arrayMin();
            // -4

Sintaxis

_arrayMin()

Descripcion

_arrayMIn() retornara el valor numerico mas chico que encuentre en un array,ya sea valor numerico de un valor alfabetico,o numero.

Ejemplos

Obtiene el menor valor ASCII


            ["a","b","c"]._arrayMin();
            // "a"
