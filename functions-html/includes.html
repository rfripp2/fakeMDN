La función includes() determina si una matriz incluye un determinado elemento, devuelve true o false según corresponda. Utiliza el algoritmo sameValueZero para determinar si se encuentra el elemento dado.

Demo

          
                var array1 = [1, 2, 3];

                console.log(fakeIncludes(array1, 2));
                // expected output: true

                var pets = ['cat', 'dog', 'bat'];

                console.log(fakeIncludes(pets,'cat'));
                // expected output: true

                console.log(fakeIncludes(pets,'at'));
                // expected output: false

Sintaxis

fakeIncludes(array, searchElement

Parámetros
array
El array que la funcion recorrera
searchElement
El elemento a buscar.

Ejemplos

Evaluar si los numeros del array cumplen con la condición

            
                var numbers = [1, 2, 3];
                fakeIncludes(numbers, 2);     // true
                fakeIncludes(numbers, 4);     // false
                fakeIncludes(numbers, (3, 3));  // false
                fakeIncludes(numbers, (3, -1)); // true
                
                var nanarray = [1, 2, NaN];
                fakeIncludes(nanarray, NaN); // true
