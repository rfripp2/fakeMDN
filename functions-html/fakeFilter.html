fakeFilter

faleFilter crea un nuevo array con los elementos que pasen la funcion proveida en el callback.

Sintaxis

fakeFilter(array,callback(elemento))

Par√°metros

  • elemento

El elemento que sera procesado por la funcion callback

Descripcion

fakeFilter llama a la funcion callback,la cual testea si da true o false para cada elemento del array. Los que pasen el test,(la funcion retorne true),seran copiados a un nuevo array,el cual fakeFilter retornara. Si ningun elemento pasa el test, fakeFilter devuelve un array vacio.

fakeFilter no muta el array
