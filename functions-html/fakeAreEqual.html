fakeAreEqual compara 2 arrays y retorna un valor booleano de acuerdo a si los arrays son estrictamente iguales.

Demo

          
            let arrA = [1, 2, 3];
            let ArrB = [1, 2, 3];
            
            fakeAreEqual(arrA, ArrB);
            // true

Sintaxis

fakeAreEqual(arr1,arr2)

Parámetros
arr1
El primero array que se comparara con arr2
arr2
El segundo array que se comparara con arr1

Descripcion

fakeAreEqual hace una comparacion estricta dos arrays,y devuelve un valor booleano de acuerdo a si son exactamente iguales.

Ejemplos

Comparar 2 arrays con iguales valores,pero diferente tipo de dato

Evaluar si los valores del array cumplen con la condición

let arrA = [1, 2, 3];
            let ArrB = [1, "2", 3];
            
            fakeAreEqual(arrA, ArrB);
            // false
